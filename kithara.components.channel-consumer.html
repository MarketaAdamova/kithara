with-channel

(with-channel consumers)(with-channel consumers channel-options)

Wrap the given consumer(s) with setup/teardown of a RabbitMQ channel. The following options can be given:

  • :channel-number
  • :prefetch-count
  • :prefetch-size
  • :prefetch-global?

If not options are given, a channel with server-side default settings will be set up.

Note: Consumers have to implement HasHandler and HasChannel.
